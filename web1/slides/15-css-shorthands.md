# Shorthand
## zkrácený zápis

---

## Jedna hodnota

```css
.foo { padding: 10px; }

.bar { 
	padding-top: 10px;
	padding-right: 10px;
	padding-bottom: 10px;
	padding-left: 10px;
}
```
<!-- .element: class="c-text-md stretch" -->

>>>
* `.foo` i `.bar` dělají to samé

---
